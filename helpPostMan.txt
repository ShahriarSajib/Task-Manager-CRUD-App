Create Tasks:
    POST → http://localhost:5000/tasks

Get/Read all tasks:
    GET → http://localhost:5000/tasks

Update tasks:
    PUT http://localhost:5000/tasks/1(task_id) 

Delete task:
    DELETE http://localhost:5000/tasks/1(task_id) 

Optionals-
 Filtering by status:
    GET /tasks?status=Completed , will be filtered to show only completed tasks

 Searching by keyword in title or description:
    GET /tasks?search=meeting , will return tasks with 'meeting' in title or description

 Sorting by createdAt or updatedAt:
    GET /tasks?sortBy=createdAt&order=asc, will sort tasks by creation date in ascending order
    GET /tasks?sortBy=updatedAt&order=desc , will sort tasks by updated date in descending order
